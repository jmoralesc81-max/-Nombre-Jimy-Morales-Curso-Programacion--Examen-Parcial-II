Descripción del programa : Este programa desarrollado permite ingresar números enteros y clasificarlos automáticamente en diferentes estructuras de datos según su tipo. Además, guarda un historial de los datos ingresados y permite construir un árbol binario de búsqueda (BST) para realizar recorridos y búsquedas.

Estructuras utilizadas : Se utilizaron varias estructuras de datos vistas en clase, como listas simplemente enlazadas, pila, cola, lista doblemente enlazada para el historial y un árbol BST. Todo fue implementado utilizando punteros y memoria dinámica con new

Explicación de la lógica : El programa recibe números ingresados por el usuario y los clasifica automáticamente: los números pares positivos se almacenan en una cola, los impares positivos en una pila y los negativos en una lista simple. Los valores válidos también se guardan en un historial usando una lista doble. Posteriormente, el usuario puede construir un árbol BST con los números positivos y realizar diferentes operaciones como recorridos, búsqueda y reorganización de estructuras.


Instrucciones de ejecución : Para ejecutar el programa, se debe compilar el archivo a través de un menú interactivo, el usuario puede ingresar datos y seleccionar las diferentes opciones disponibles para visualizar, buscar o procesar la información almacenada.
